# Anleitung für Cloudflare Pages + Resend Setup

1. Dieses Projekt bei GitHub hochladen
2. In Cloudflare Pages ein neues Projekt aus diesem Repository erstellen
3. Unter "Settings" → "Environment Variables":
   - `RESEND_API_KEY` setzen auf deinen API-Schlüssel
4. Seite veröffentlichen
5. Kontaktformular ausfüllen → E-Mail wird an moritsweba@gmail.com gesendet

Hinweis: Absender ist aktuell onboarding@resend.dev (Testmodus). 
Nach Domain-Verifizierung bei Resend kannst du kontakt@mediamw.de o.ä. nutzen.
